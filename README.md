# GOV-1.0
This repository contains the R code for the main simulation results in: Dafermos Y. and Nikolaidi, M. (2026). Ecological stock-flow consistent modelling: a theoretical and practical guide, in S., D’Alessandro, T., Distefano and G., Morlin (eds.), Handbook of Ecological Macroeconomics, Open Book Publishers, forthcoming

To run this code: (1) install R and RStudio; (2) click on the 'Code' button and download and extract the ZIP folder; (3) open the .R file in RStudio and run the code by clicking on the 'run' button.

For more information about the DEFINE-GOV model, see here https://define-model.org. For any questions or comments, please write to: yannis.dafermos@soas.ac.uk or m.nikolaidi@greenwich.ac.uk.
